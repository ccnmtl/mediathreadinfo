Summer 2016 Updates
September 1, 2016
Columbia courses can now self-register their courses each semester(Columbia University only). Instructors in each course can now manage members in their course by promoting them to instructors or downgrading them to students. Instructors can add individuals to their course using a single or bulk adding users. The Mediathread login experience has been improved with a more organized course list view and home navigation.
Spring 2016 Updates to Mediathread
May 31, 2016
The Mediathread team upgraded the composition editor using TinyMCE to allow for a better editing experience. The application updates contain improvements on web interoperability and works better with various web media sites like Vimeo. As well, the tool is now LTI compliant.
